Window イベント

Window 型は下記のメンバーを公開します。

イベント
  名前説明
Closed
メッセージループの終了時に実行されます。
Destroyed
ウィンドウハンドルが失効する直前に実行されます。
Started
メッセージループの開始時に実行されます。
Updated
メッセージループの度に実行されます。
