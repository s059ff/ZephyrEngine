CollisionComponent メソッド

CollisionComponent 型は下記のメンバーを公開します。

メソッド
  名前説明
OnAttach
ワールド空間に物体を登録します。
(EntityComponentOnAttachをオーバーライド)
OnDestroy
このコンポーネントを所持しているエンティティが破棄されるときに呼び出されます。リソースの解放処理を行います。このメソッド内で親エンティティにアクセスすることはできません。
(EntityComponentから継承)
OnDetach
ワールド空間から登録を解除します。
(EntityComponentOnDetachをオーバーライド)
ReceiveMessage (EntityComponentReceiveMessage(Object, Object)をオーバーライド)
SendMessage
外部コンポーネントにメッセージを送信します。
(EntityComponentから継承)
