StreamingSoundBuffer メソッド

StreamingSoundBuffer 型は下記のメンバーを公開します。

メソッド
  名前説明
Close
サウンドデータを破棄し、サウンドファイルを閉じます。
Create
サウンドファイルを開いて、サウンドを再生可能な状態にします。
Dispose (INativeWrapper<zephyr::sound::StreamingSoundBuffer>から継承)
Dispose(Boolean) (INativeWrapper<zephyr::sound::StreamingSoundBuffer>から継承)
Finalize (INativeWrapper<zephyr::sound::StreamingSoundBuffer>から継承)
LoopPlay
サウンドを先頭からループ再生します。
Pause
サウンドを一時停止もしくは再開します。
Play
サウンドを先頭から再生します。
Seek
秒単位で再生位置を変更します。
Stop
サウンドを停止し、再生位置を先頭に戻します。
Update
再生状況に応じて、サウンドをメモリに読み込みます。このメソッドは毎フレーム呼び出す必要があります。
