SoundBuffer プロパティ

SoundBuffer 型は下記のメンバーを公開します。

プロパティ
  名前説明
Duration
サウンドの長さを秒単位で取得します。
Frequency
サンプリング周波数を Hz 単位で取得または設定します。
NativePtr (INativeWrapper<zephyr::sound::SoundBuffer>から継承)
OriginalFrequency
原音のサンプリング周波数を取得します。
Pan
パンを -1 ～ +1 の範囲で取得または設定します。 -1 のとき左側のスピーカーからのみ音が鳴り、 +1 のとき右側のスピーカーからのみ音が鳴ります。
Playing
サウンドがを再生中か調べます。
Stopped
サウンドが停止中か調べます。
Volume
音量を 0 ～ 1 の範囲で取得または設定します。 0 のとき最小、 1 のとき最大を表します。
