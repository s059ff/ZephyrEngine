CollisionComponent プロパティ

CollisionComponent 型は下記のメンバーを公開します。

プロパティ
  名前説明
Excludes
衝突検出から除外する対象のリストを設定します。
Group
自分の衝突検出グループを設定します。
HasOwner
コンポーネントがエンティティに取り付けられているか調べます。
(EntityComponentから継承)
Object
衝突オブジェクトを設定します。
OtherGroups
衝突検出を行う相手の衝突検出グループを設定します。
Owner
コンポーネントを所持しているエンティティ。
(EntityComponentから継承)
Penetration
このプロパティが true の場合は、一フレーム中に複数の物体と衝突が発生します。
