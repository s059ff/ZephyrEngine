ウィンドウをアクティブにします。
Close
メッセージループを終了し、ウィンドウを閉じます。

ウィンドウを作成します。

HideFrame

SetPosition
ウィンドウの位置を変更します。
SetSize
クライアント領域の大きさを変更します。
ShowCursor
カーソルを表示します。
ShowFrame
フレームを表示します。
Start
メッセージループを開始します。ウィンドウが閉じられるまで、この関数は終了しません。
