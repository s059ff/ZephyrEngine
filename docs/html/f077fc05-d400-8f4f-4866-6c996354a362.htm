ProjectorComponent メソッド

ProjectorComponent 型は下記のメンバーを公開します。

メソッド
  名前説明
OnAttach
コンポーネントがエンティティにアタッチされたときに呼び出されます。
(EntityComponentから継承)
OnDestroy
このコンポーネントを所持しているエンティティが破棄されるときに呼び出されます。リソースの解放処理を行います。このメソッド内で親エンティティにアクセスすることはできません。
(EntityComponentから継承)
OnDetach
コンポーネントがエンティティからデタッチされたときに呼び出されます。
(EntityComponentから継承)
ReceiveMessage
外部コンポーネントからのメッセージを受信して処理を行います。
(EntityComponentから継承)
SendMessage
外部コンポーネントにメッセージを送信します。
(EntityComponentから継承)
SetOrthogonalMode
平行投影モードを設定します。
SetPerspectiveMode
透視投影モードを設定します。
